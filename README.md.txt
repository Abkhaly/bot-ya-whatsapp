# WhatsApp Group Bot - Zanzibar Mapishi Edition

## Features
- Kufuta link na picha zisizofaa
- Onyo la kwanza kwa matangazo, onyo la mwisho na kumtoa mtu group
- Kujibu salamu za kawaida
- Kukaribisha member mpya na ujumbe wa mapishi Zanzibar
- Kutuma ujumbe wa asubuhi njema na motisha kila asubuhi saa 8:00 AM

## Jinsi ya kuendesha
1. Install dependencies: `npm install`
2. Run bot local: `node index.js`
3. Scan QR code mara ya kwanza tu
4. Upload `.wwebjs_auth` folder na files zote kwenye GitHub
5. Deploy kwenye Render na furahia bot isiyohitaji QR tena
